# Frontend

This folder will contain the React application for the user-facing dashboard.

Planned contents:

- Pages and reusable UI components.
- API client code for talking to the FastAPI backend.
- State management for forecasts, filters, and charts.
- Styling, assets, and frontend build configuration.

Primary role:

The frontend will visualize air-quality forecasts, historical trends, model confidence, and operational status exposed by the backend.
